# Alarm Clock Program

A Python alarm clock that continuously checks the current time and plays an audio file when the alarm time matches.

## Features
- Time checking using `datetime`
- Plays `.mp3` sound using `pygame`
- Runs until alarm is triggered and sound finishes

## Requirements
- Python 3
- `pygame` library

## Concepts Used
- `time` module
- `datetime` module
- `pygame.mixer`
- Loop-based scheduling
